필요한 기능 정리 및 레이아웃 초안 작성
1. 메인 페이지
① 검색, 랭킹, 서비스 예정작, 종료 예정작, 하단바(홈, 탐색, 마이페이지) 필요

1-1 검색
설명 : 검색 바 누르면 이동하는 페이지로, 작품을 검색하는 페이지
① 돌아가기, 검색, 하단바(홈, 탐색, 마이페이지) 필요
② 검색 후 작품 누르면 정보 페이지로 이동

1-2 랭킹
설명 : 랭킹 옆 > 누르면 이동하는 페이지로, 플랫폼 별 인기 순위를 나타내는 페이지
① 돌아가기, 검색, 플랫폼 별 필터, 작품 나열, 하단바(홈, 탐색, 마이페이지) 필요
② 작품 누르면 정보 페이지로 이동

1-3 서비스 예정작
설명 : 서비스 예정작 옆 > 누르면 이동하는 페이지로, 플랫폼 별 서비스 예정작을 날짜순으로 나타내는 페이지
① 돌아가기, 검색, 플랫폼 별 필터, 날짜식으로 작품 나열, 하단바(홈, 탐색, 마이페이지) 필요
② 작품 누르면 정보 페이지로 이동 

1-4 종료 예정작
설명 : 종료 예정작 옆 > 누르면 이동하는 페이지로, 플랫폼 별 서비스 종료 예정작을 날짜순으로 나타내는 페이지
① 돌아가기, 검색, 플랫폼 별 필터, 날짜식으로 작품 나열, 하단바(홈, 탐색, 마이페이지) 필요
② 작품 누르면 정보 페이지로 이동

1-5 하단바(홈)
① 누르면 메인 페이지로 이동

1-6 하단바(탐색)
① 누르면 탐색 페이지로 이동

1-7 하단바(마이페이지)
① 누르면 마이 페이지로 이동

2. 탐색 페이지
설명 : 플랫폼 별 작품 표기하는 페이지
① 검색, 플랫폼 별 필터, 작품 필터, 작품 표기, 하단바(홈, 탐색, 마이페이지) 필요
② 작품 누르면 정보 페이지로

3. 정보 페이지
설명 : 해당 콘텐츠의 정보를 나타내는 페이지
① 돌아가기, 검색, 찜하기, 서비스 플랫폼, 정보, 하단바(홈, 탐색, 마이페이지) 필요

4. 마이 페이지
설명 : 찜한 작품의 스케줄을 달력 형식으로 작성해주는 페이지
