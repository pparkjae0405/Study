# C# 프로그래밍 시작하기

변수 개념 잡기
```
- 변수
값을 저장하는 장소
```

함수(메서드) 개념 잡기
```
- 함수
미리 지정된 동작을 수행하는 코드 묶음
```

첫 스크립트 작성
```
1. Project > C# Script > 이름변경 > 코드 작성
2. Hierarchy > Create > Create Empty
3. 스크립트를 GameObject로 드래그&드롭
4. 실행 > 콘솔창 확인
```

코딩 기본 규칙
```
- 주석
//나 /**/로 사용

- 세미콜론
코드 한 문장이 끝나면 ;를 붙임

- using
자바의 import 역할
```

변수 연습하기
```
자료형 이름 = 값 선언; 으로 사용
```

메서드 연습하기
```
함수를 선언한 뒤 메인 함수에서 호출하여 사용
```

제어문
```
if, if-else, 논리 연산자(&&, ||, !) 사용
```

for문
```
for(초기화 ; 조건 ; 갱신) 으로 사용
```

while문
```
while(조건) 으로 사용
```

배열
```
int[] 이름 = new int[배열의 크기]; 로 선언
```
