# 닷지 플레이어 제작

머터리얼(Material)
```
셰이더와 텍스처가 합쳐진 에셋으로, 오브젝트의 픽셀 컬러를 결정

- Albedo(반사율)
물체가 어떤 색을 반사할지 결정하는 것으로, 물체 표면의 기본색 결정

- 사용법
Material > 색상 변경 > 게임 오브젝트로 드래그&드롭
```

태그 설정
```
Inspector > Tag > Player 선택
나중에 탄알을 만들 때 충돌한 오브젝트가 플레이어인지 
확인하기 위해 Player 태그를 사용
````

입력 감지
```
- Update() 메서드
한 프레임에 한번 번, 매 프레임마다 반복 실행

- Input 클래스
사용자 입력을 감지하는 메서드를 모아둔 집합

- Die() 메서드
자신의 게임 오브젝트를 비활성화하여 죽음을 구현하는 메서드

- gameObject
자신이 추가된 게임 오브젝트를 가리키는 변수

- SetActive() 메서드
게임 오브젝트를 활/비활성화하는 버튼으로,
Inspector 창에서 게임 오브젝트 이름 왼쪽의 체크박스
```

개선
```
- GetComponent() 메서드
원하는 타입의 컴포넌트를 자신의 게임 오브젝트에서 찾아오는 메서드
```
