```
GoF(Gangs of Four) 디자인 패턴
- factory method pattern : 상위 클래스에서 객체를 생성하는 인터페이스 정의,
  하위 클래스에서 인스턴스 생성
- prototype pattern : prototype 먼저 생성, 인스턴스 복제하여 사용
- bridge pattern : 구현부에서 추상층을 분리하여 각자 독립적으로 확장 가능
- Adapter pattern : 기존에 구현되있는 클래스에 기능 발생 시 클래스를 
  재사용할 수 있도록 중간에서 맞춰주는 역할
- mediator pattern : 객체 간 통제, 지시의 역할을 하는 중재자를 두어 객체지향 목표 달성
```

```
- MBR(memory buffer register) : 기억장치 출입 데이터가 잠시 기억되는 레지스터
- NAC(network access control) : PC의 MAC주소 IP관리시스템에 등록
- NIC : 물리 계층과 데이터 계층의 서비스 제공
- SAN : 다른 운영체제를 가진 여러 기종들이 네트워크상에서 동일 저장장치의 데이터를
  공유하게 함으로써, 여러 개의 저장장치나 백업 장비를 단일화시킨 시스템

SSH(Secure Shell)
- 기본 네트워크 포트는 22번
- 전송되는 데이터는 암호화
- 키를 통한 인증을 클라이언트의 공개 키를 서버에 등록해야 함
- 서로 연결되어있는 컴퓨터 간 원격 명령 실행, 셀 서비스 수행

- Bell-Lapadula Model : 군대의 보안레벨처럼 정보의 기밀성에 따라 정보 보호

- 스쿱(Sqoop) : 하둡(Hadoop)과 관계형DB 간 데이터 전송 도구
```
