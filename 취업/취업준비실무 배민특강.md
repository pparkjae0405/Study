# 취업준비실무 배민특강

특강 내용
```
IT기업이 top50 기업 중 대다수를 차지하고 있을 만큼 중요한 위치이다.
높은 연봉, 워라밸, 편안한 근무환경

본인이 어떤 점에서 강점이 있는지 생각해보고,
코딩테스트를 등한시하지말고 준비했으면 좋겠다.

그 상황에 필요한 인력이 뽑힐 가능성이 높으므로 
실망하지말고 많이 지원하고 면접을 보는 게 중요하다.

관련된 사람에게 질문이나 연락을 적극적으로 해 보자.

면접에서는 본인이 했던 것을 얼마나 잘 설명하고 
깊게 알고 있는지를 질문하는 경향이 있는데,
평소에 공부하며 계속 의문을 가지면 이런 질문들을 대비할 수 있다.
ex) 백엔드 웹 개발자를 목표로 스프링 부트를 공부하고 있다면 
백엔드 웹 개발자가 어떤 일을 한다고 생각하며, 왜 이걸 하고 싶은가?
스프링 부트를 사용해봤는데, 스프링과 스프링부트의 차이점은?
왜 스프링 부트를 선택하셨는지?
다른 소프트웨어는 어떤 것을 사용해봤는지? 등

블라인드 사이트를 활용하여 기업 조사가 가능하다.
```
